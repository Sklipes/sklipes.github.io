# Solution: Fine Arts

### DONALD JUDD

Each of these "art pieces" are words encryptid with common codes. Decrypting all of them will get you the clue phrase `SPECIFIC OBJECTS ART STYLE CREATOR SIX FOUR` which clues the answer **DONALD JUDD**.

|Code|Translation|
|:----:|:----:|
|Morse|SPECIFIC|
|Elements|OBJECTS|
|Braille|ART STYLE|
|Binary|CREATOR|
|Semaphore|SIX FOUR|
